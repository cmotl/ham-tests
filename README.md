# Ham Tests

## Purpose

- Provide a customized study and test plan covering amateur radio Technician, General, and Extra exams.

## Build the app

```
> ./interactive.sh
$ npm build
```

## Test the app

```
> ./interactive.sh
$ npm test
```

## Start interactive webserver

```
> ./interactive.sh
$ npm start
```

Then open a webbrowser to http://localhost:3333
